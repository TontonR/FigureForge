Termes i Condicions

Objecte

Aquests Termes i Condicions regulen l’ús del lloc web Figure Forge, creat exclusivament amb finalitats educatives com a projecte escolar. L’accés i ús del lloc web impliquen l’acceptació d’aquests termes.

Ús del lloc web

L’usuari es compromet a utilitzar el lloc web de manera responsable, respectant la legislació vigent i sense realitzar activitats que puguin danyar el funcionament del projecte o el seu contingut demostratiu.

Contingut i finalitat educativa

Tot el contingut del lloc web, incloent productes, formularis, registres i compres simulades, té una finalitat exclusivament educativa i no representa una activitat comercial real.

Registre d’usuaris

En cas que el lloc web permeti el registre d’usuaris, les dades introduïdes són simulades i s’utilitzen únicament per demostrar el funcionament d’un sistema web dins del projecte educatiu.

Propietat intel·lectual

Tots els continguts del lloc web (textos, imatges, logotips i disseny) pertanyen al projecte Figure Forge o s’utilitzen amb finalitats educatives. No està permesa la seva reproducció amb finalitats comercials.

Responsabilitat

Figure Forge no es fa responsable de possibles errors tècnics ni de l’ús indegut que puguin fer els usuaris del lloc web, atès que es tracta d’un entorn de demostració.

Enllaços externs

El lloc web pot incloure enllaços a pàgines externes. Figure Forge no té control sobre aquests llocs i no es responsabilitza del seu contingut ni de les seves polítiques.

Modificacions

Figure Forge es reserva el dret de modificar aquests Termes i Condicions en qualsevol moment per adaptar-los a les necessitats del projecte educatiu.

Acceptació dels termes

L’ús del lloc web implica l’acceptació plena d’aquests Termes i Condicions.
